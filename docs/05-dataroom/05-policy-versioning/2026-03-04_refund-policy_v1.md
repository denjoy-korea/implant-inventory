## 요약

| 구분 | 기준 | 환불 방식 |
|------|------|---------|
| 전액 환불 | 결제 후 7일 이내 + 이용 내역 없음 | 100% 전액 |
| 중도 환불 | 이용 개시 후 | 일할 계산 후 잔여분 |
| 연간 구독 중도 해지 | 연간 할인 적용 취소 | 월정가 기준 재산정 후 잔여분 |
| 환불 불가 | 사용 완료 구간 | 해당 없음 |

---

## 상세 환불 규정

### 1. 전액 환불 조건

**[핵심 조건]** 다음 두 가지를 모두 충족하는 경우 전액 환불:

- 결제일로부터 **7일 이내**
- 서비스 이용 내역(데이터 입력, 조회 등) **없음**

### 2. 중도 환불 산정 방식

서비스 이용이 개시된 이후:

```
환불 금액 = 결제 금액 - (일할 단가 × 이용 완료 일수)
일할 단가 = 결제한 플랜의 정가 ÷ 해당 구독 기간의 총 일수
```

### 3. 연간 구독 중도 해지

1. 연간 구독 할인 혜택 **취소**
2. 환불 신청일까지 이용 기간 → **월간 구독 정가** 기준 재산정
3. `환불 금액 = 연간 결제금액 - (월정가 × 이용 월수)`

### 4. 환불 제한

디지털 서비스의 특성상 아래 경우 청약철회 또는 환불이 **제한**될 수 있습니다:
- 사용이 개시된 구간
- 제공이 완료된 구간

(관련 법령이 허용하는 범위 내 적용)

### 5. 환불 처리 기간

환불 신청 접수 후 실제 처리: **영업일 기준 5~7일** (카드사 및 결제망 상황에 따라 상이)

### 6. 해지 방법

- 서비스 설정 화면 → 구독 관리 → 해지 신청
- 고객지원 채널: admin@denjoy.info

## 무료 체험 관련

- 무료 체험 기간에는 결제가 발생하지 않으므로 환불 대상 아님
- 체험 종료 후 15일 이내 유료 전환하지 않으면 데이터 자동 삭제
- 베타 신청자: 28일 체험 / 일반: 14일 체험
