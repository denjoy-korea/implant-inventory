## 계약 모델

DenJOY는 웹 기반 SaaS로, **클릭-스루 방식(Click-Through Agreement)** 으로 계약이 체결됩니다.
- 회원가입 및 플랜 결제 시 이용약관에 동의 체크 → 계약 성립
- 별도 오프라인 서면 계약은 기업 고객 맞춤 도입 시에만 적용 예정
