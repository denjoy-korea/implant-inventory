### P0/P1 사고 발생 시

```
1. 탐지 → 슬랙/메일 알림 즉시 발송
2. 초기 대응 → 영향 범위 파악 (15분 내)

6. 후속 조치 → 재발 방지책 수립, 포스트모템 작성
```
