# Plan: value-reposition

## 개요

**목표**: `ValuePage.tsx`를 "엑셀 재고관리 도구" 프레임에서 **"치과 운영 인텔리전스 플랫폼"** 으로 리포지셔닝

**배경**: LandingPage 리포지셔닝(landing-reposition) 완료 후, 동일한 가치 전달 갭이 ValuePage에도 잔존. ValuePage는 랜딩에서 "더 알아보기" 흐름으로 진입하는 심층 설득 페이지이므로, 랜딩과 일관된 메시지 체계를 유지해야 함.

**대상 파일**: `components/ValuePage.tsx` (478행)

---

## 수정 계획

### Phase 1: Hero 재구성

#### 1-1. 배지 + 헤드라인
- **배지**: `"지금 이 순간에도 시간이 낭비되고 있습니다"` → `"병원 데이터가 일하게 하세요"`
- **배지 색상**: `rose-500` → `indigo-400` (위기 → 기회 프레임)
- **헤드라인**:
  - 현재: `"아직도 엑셀로 임플란트 재고를 관리하시나요?"`
  - 변경: `"수술 기록이 쌓일수록,"` + 그래디언트: `"병원 운영이 똑똑해집니다"`
- **서브카피 1**: 제거 (`"매주 2시간, 연간 104시간을..."`)
- **서브카피 2**: `"재고 차감, 발주 추천, 교환 추적, 임상 분석까지 — 수술 기록 하나가 모든 것을 움직입니다."` (1개로 통합)

### Phase 2: Pain Points 4개 재구성

**섹션 타이틀**: `"엑셀 재고관리가 만드는 숨은 손실"` → `"지금 이 순간, 병원에서 일어나고 있는 일"`

Pain Points 4개:

1. **임상 품질 사각지대**
   - 제목: `"이 브랜드, 왜 계속 교환이 나올까?"`
   - 설명: `교환이 반복되는데 어떤 제조사, 어떤 규격에서 문제가 나는지 데이터가 없어 패턴을 파악할 수 없는 상황`
   - 지표: `임상 의사결정 근거 부재`

2. **재고 관리 악순환**
   - 제목: `"실물이랑 시스템이 또 다르네"`
   - 설명: `수동 관리로 오차가 누적되고, 필요한 건 없고 남는 건 쌓이는 악순환. 실사할 때마다 불일치가 당연해진 상황`
   - 지표: `재고 정확도 저하`

3. **경영 데이터 부재**
   - 제목: `"우리 병원 월 수술 건수가 몇 건이지?"`
   - 설명: `월별 수술 건수, 브랜드별 점유율, 연도별 추세 — 경영 판단에 필요한 기초 데이터조차 정리되지 않은 상황`
   - 지표: `데이터 기반 경영 불가`

4. **직원 업무 피로** (기존 유지, 표현 정제)
   - 제목: `"이 발주, 내가 또 확인해야 해?"`
   - 설명: `수술 직전 재고 부족 발견, 긴급 발주 반복으로 인한 업무 피로. 시스템 부재로 모든 책임이 사람에게 쏠리는 구조`
   - 지표: `업무 부담 집중`

### Phase 3: Before/After 재구성

#### 3-1. Before 컬럼 타이틀 변경
- `"엑셀 수동 관리"` → `"데이터 없는 운영"`

#### 3-2. Before 항목 (6개) 교체
1. `"교환 패턴을 제조사별로 파악할 수 없음"` (임상)
2. `"월별 수술 건수와 브랜드 점유율 데이터 없음"` (경영)
3. `"재고 파악에 30분~1시간 소요"` (운영, 유지)
4. `"수술 기록과 재고가 따로 관리"` (유지)
5. `"발주 시점을 감으로 판단"` (유지)
6. `"교환 이력 추적 어려움"` (유지)

#### 3-3. After 컬럼 타이틀 변경
- `"DenJOY 자동 관리"` → `"데이터가 일하는 운영"`

#### 3-4. After 항목 (6개) 교체
1. `"FAIL 자동 감지 + 제조사별 교환율 실시간 분석"` (임상)
2. `"수술 트렌드, 브랜드 점유율, 임상 히트맵 대시보드"` (경영)
3. `"클릭 한 번으로 전체 재고 현황 확인"` (유지)
4. `"수술 기록 업로드 시 재고 자동 차감"` (유지)
5. `"데이터 기반 적정 재고량 자동 계산"` (유지)
6. `"교환 접수부터 입고까지 자동 추적"` (유지)

#### 3-5. Aha! Moment 카드 교체
