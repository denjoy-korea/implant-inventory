# Plan: DenJOY 제품 분석 및 비즈니스 전략

> 임플란트 재고관리 SaaS **DenJOY(덴조이)**의 현황을 진단하고,
> 한국 치과 시장에서 지속 성장 가능한 비즈니스 전략을 수립한다.

---

## 1. 제품 현황 분석 (As-Is)

### 1.1 제품 정의

| 항목 | 내용 |
|------|------|
| 제품명 | DenJOY (덴조이) |
| 카테고리 | B2B SaaS — 치과 임플란트 재고·수술 관리 |
| 핵심 가치 제안 | **"연 104시간을 돌려드립니다"** (주 2시간 절감) |
| 타겟 시장 | 대한민국 치과의원 (임플란트 시술 비중 높은 클리닉) |
| 기술 스택 | React 19 + TypeScript + Supabase + PWA |

### 1.2 핵심 기능 목록

| 기능 | 설명 | 플랜 |
|------|------|------|
| 재고 관리 마스터 | 품목별 재고 추적, 부족분 자동 감지 | All |
| 재고 실사 | 실물 vs 시스템 대조, 불일치 기록 | All |
| 수술 기록 DB | 식립·FAIL 데이터 업로드 + 통계 대시보드 | All |
| 식립 FAIL 관리 | FAIL 픽스쳐 추적, 브랜드별 교환 발주 | All |
| 주문 관리 | 원클릭 발주, 발주 이력 관리 | Business+ |
| 역할 관리 | 원장·직원·스태프 권한 분리 | Plus+ |
| 감사 로그 | 재고 변경 이력 전체 기록 | Business+ |
| PWA + 모바일 | 스마트폰에서 재고 조회·수정 | All |

### 1.3 현재 수익 구조

| 플랜 | 월정액 | 연정액 | 용량 | 멤버 | 현재 가입 수 |
|------|--------|--------|------|------|------------|
| Free | ₩0 | ₩0 | 100품목 | 1명 | 2개 (2%) |
| Basic | ₩29,000 | ₩23,000/월 | 200품목 | 3명 | 0개 |
| Plus | ₩69,000 | ₩55,000/월 | 500품목 | 5명 | 0개 |
| Business | ₩129,000 | ₩103,000/월 | 무제한 | 무제한 | 2개 (내부) |
| **합계 MRR** | | | | | **₩0 (추정)** |

> **진단**: 현재 유료 전환이 전무하며, 제품은 PMF(Product-Market Fit) 검증 전 단계.
> Free 플랜 2개소 + 내부(Ultimate) 2개소로 총 4개 워크스페이스 운영 중.

### 1.4 강점 (Strengths)

| 강점 | 세부 내용 |
|------|---------|
| **도메인 특화** | 임플란트 플로우(식립 → FAIL → 교환 → 발주)에 최적화된 유일한 SaaS |
| **데이터 깊이** | 수술기록 × 재고 × FAIL 상관관계 분석 가능 — 경쟁 제품 없음 |
| **현대적 UX** | React 19 + Tailwind, PWA, 모바일 최적화로 레거시 솔루션 대비 압도적 UX |
| **보안** | 환자정보 암호화(SubtleCrypto), 감사 로그, Supabase RLS |
| **실시간** | Supabase Realtime으로 멀티유저 동시 작업 지원 |

### 1.5 약점 (Weaknesses)

| 약점 | 세부 내용 |
|------|---------|
| **수동 업로드 의존** | 수술기록을 엑셀로 수동 업로드 — 자동화 없음 |
| **온보딩 마찰** | 기초재고 설정, 엑셀 형식 이해 필요 → 초기 이탈 위험 |
| **영업 채널 부재** | D2C(랜딩페이지) 외 파트너 채널 없음 |
| **AI 기능 미완성** | AI 수요 예측이 Business 플랜 표기만 있고 실제 미구현 |
| **결제 연동 UX** | 수동 계좌이체 방식으로 SaaS 즉시 결제 불가 |

### 2.2 경쟁 구도

| 경쟁자 | 유형 | 약점 | DenJOY 우위 |
|--------|------|------|------------|
| 덴트웹(Dentweb) | 치과 EMR/POS | 재고·FAIL 기능 없음 | 임플란트 특화 깊이 |
| 이지웰페어 | 병원 EMR | 복잡, 고비용, 비모바일 | 가격, UX, 속도 |
| Excel + 수기 장부 | 무도구 | 비효율, 오류, 협업 불가 | 자동화, 분석, 협업 |
| Osstem 자사 도구 | 브랜드 단일 | 단일 브랜드만 지원 | 멀티브랜드 |

> **결론**: 한국 치과 임플란트 재고관리 SaaS 분야는 **사실상 Red Ocean이 없는 블루오션**.
> 지금이 선점 타이밍.

## 3. 비즈니스 전략 (To-Be)

### 3.1 전략 방향 (Strategic Bets)

```
[핵심 전략]: "임플란트 클리닉의 운영 두뇌가 된다"
Become the Operating System for implant-focused dental clinics in Korea
```

**3가지 전략 기둥:**

| 기둥 | 방향 | 기간 |
|------|------|------|
| **① 제품-시장 검증** | Free → Paid 전환 첫 10개소 확보 | 0-3개월 |
| **② 채널 확장** | 거래처(딜러)·학회 파트너십으로 공급 루트 다각화 | 3-9개월 |
| **③ 데이터 네트워크** | 익명 집계 벤치마크로 네트워크 효과 → 이탈 방지 | 9-18개월 |

### 3.2 Phase 1 — PMF 검증 (0~3개월)

**목표**: 유료 전환 10개소, MRR ₩500K 달성

| 과제 | 실행 방법 |
|------|---------|
| **온보딩 단축** | 1-Click 기초재고 템플릿 제공 (Osstem/IBS/Dentium 표준셋) |
| **결제 자동화** | 토스페이먼츠·나이스페이 연동으로 카드 즉시 결제 |
| **Free → Basic 넛지** | 3개월 데이터 보관 만료 7일 전 알림 + 업그레이드 CTA |
| **레퍼런스 구축** | 헤비유저 2~3개소 심층 인터뷰 → 케이스스터디 제작 |
| **Support SLA** | 24시간 내 카카오채널 응대 체계 구축 |

**성공 지표 (KPI)**:
- Activation Rate (기초재고 50% 이상 입력 완료): > 60%
- Free → Paid 전환율: > 15%
- 30일 Retention: > 70%

### 3.3 Phase 2 — 채널 확장 (3~9개월)

**목표**: 유료 50개소, MRR ₩3.5M

| 채널 | 실행 방법 | 기대 효과 |
|------|---------|---------|
| **임플란트 딜러 파트너** | 지역 딜러사에 수수료 10% 리셀러 모델 제안 | 한 딜러당 5~10개소 |
| **학회 스폰서십** | 대한구강악안면임플란트학회(KAOMI), KDA 부스 | 브랜드 인지도 + 리드 |
| **KOL 전략** | 인스타 팔로워 많은 원장 3~5명에게 Business 무료 제공 → 후기 | 바이럴 효과 |
| **덴트웹 교체 캠페인** | "덴트웹에서 DenJOY로" 전환 가이드 + 데이터 마이그레이션 | 기존 시장 흡수 |

### 3.4 Phase 3 — 데이터 해자 구축 (9~18개월)

**목표**: 유료 200개소, MRR ₩14M, NRR > 110%

| 전략 | 내용 |
|------|------|
| **익명 벤치마크** | "전국 Plus 플랜 치과 평균 FAIL률: 3.2%" — 내 클리닉 대비 표시 |
| **AI 수요 예측 (실제 구현)** | 과거 수술 패턴 기반 2주 후 발주 필요 품목 자동 추천 |
| **원클릭 발주 네트워크** | 주요 딜러사 API 연동으로 앱 내 실발주 처리 → 커미션 수익 |
| **멀티 클리닉** | 네트워크 치과(2개소+) 통합 대시보드 — 새 플랜 추가 |
| **보험청구 연동** | 건강보험 청구 데이터와 수술기록 자동 매칭 제안 |

### 4.1 현재 → 목표 수익 구조

| 수익원 | 현재 | Phase 2 | Phase 3 |
|--------|------|---------|---------|

| **합계 MRR** | **₩0** | **₩3.5M** | **₩17M** |

### 4.2 플랜 구조 개선안

현재 플랜 구조의 문제: **Basic(₩29K)과 Plus(₩69K) 사이 가격 갭이 너무 큼**

```
[개선 방향]
Free (₩0) → Starter (₩19K) → Basic (₩39K) → Plus (₩79K) → Business (₩149K)
               ↑ 신규 추가                  ↑ 소폭 인상
```

| 변경 | 이유 |
|------|------|
| Starter ₩19,000 신설 | Free → 유료 심리 장벽 낮춤 (₩19K는 "커피 한 잔" 프레임) |
| Basic ₩39,000 (↑₩10K) | 3멤버 + 6개월 + 추가기능으로 가치 강화 |
| 연간 할인 20% → 25% | 연납 유도 강화 (현금흐름 안정) |

## 6. Go-to-Market 전략

### 6.1 ICP (이상적 고객 프로파일)

```
[Primary ICP]
- 개원 3년+ 치과의원
- 임플란트 월 30건 이상 시술
- 현재 Excel 또는 수기 재고관리
- 직원 2~4명 (치위생사 포함)
- 원장이 디지털 도구에 열린 편 (40대 이하 또는 SNS 활동)
- 서울/경기 또는 광역시 소재
```

### 6.2 고객 획득 채널 우선순위

| 순위 | 채널 | 이유 |
|------|------|------|
| 1 | **임플란트 딜러 리셀러** | 기존 신뢰관계, 수백 개소 커버 |
| 2 | **카카오/유튜브 광고** | 치과 원장 타겟팅 가능 |
| 3 | **치과 학회/세미나** | 의사결정자 직접 접촉 |
| 4 | **구글 SEO** (임플란트 재고관리) | 장기 인바운드 |
| 5 | **KOL 원장 후기** | 동료 추천 효과 |

### 6.3 전환 퍼널

```
인지 → 관심 → 가입(Free) → Activation → 전환(Paid) → 확장(Upgrade)

핵심 지점:
- 가입→Activation: 온보딩 위자드 (기초재고 설정 완료)
- Activation→전환: 14일 체험 만료 전 1:1 데모 콜 제안
- 전환→확장: FAIL 관리 데이터 100건 돌파 시 Upgrade 넛지
```
